# case-study-bright-coffee

THIS PROJECT FOCUSES ON ANALYZING CUSTOMER SALES DATA. THE GOAL WAS TO IDENTIFY KEY TRENDS,PATTERNS, AND INSIGHTS THAT CAN HELP IMPROVE DECISION MAKING.

TO COMPLETE THIS CASE STUDY, I FIRST COLLECTED THE DATASET IN CSV FORMAT. I THEN CLEANED AND PROCESSED THE DATA USING SQL AND EXCEL. AFTER PREPARING THE DATA, I PERFORMED ANALYSIS USING PIVOT TABLES AND VISUALIZATIONS TO UNCOVER MEANINGFUL INSIGHTS.

TOOLS USED:
SQL- FOR DATA CLEANING
MICROSOFT EXCEL- FOR ANALYSIS AND VISUALIZATIONS
DATABRICKS
MIRO - FOR PLANNING
CANVA- FOR PRESANTION
